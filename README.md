# 📖 소설 연재 사이트

## 🔎 프로젝트 소개
- 웹 소설 작가를 위한 연재 공간입니다.
- 이곳에서 소설을 연재하여 본인의 작품에 대한 독자 반응도를 체크할 수 있습니다.
- 응원이 일정 이상되었을 경우 다음 회차를 무료로 열람할 수 있습니다.
- 독자(회원)들은 자신이 응원하는 작품을 홍보하여 무료로 열람할 수 있습니다.

<br>
